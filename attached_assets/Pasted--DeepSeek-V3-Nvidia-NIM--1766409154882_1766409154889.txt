نص الخطة (Copy & Paste):
المهمة: تحويل هذا المشروع إلى "مساعد برمجي تلقائي" (Autonomous Coding Agent) يستخدم نموذج DeepSeek-V3 عبر Nvidia API لتعديل وإنشاء ملفات المشروع فوراً.
المطلوب منك تنفيذه:
 * بناء واجهة تفاعل: أنشئ ملف main.py يستخدم الكود المزود (Nvidia OpenAI Client) لإنشاء حلقة تكرارية (Loop) تستقبل طلباتي البرمجية.
 * نظام إدارة الملفات: أضف وظيفة (Function) برمجية تقوم بمسح (Scan) ملفات المشروع الحالية وإرسال أسمائها ومحتواها للنموذج عند كل طلب، ليكون لديه سياق كامل عن المشروع.
 * التعديل التلقائي: برمج المساعد بحيث إذا طلب منه "إنشاء ملف" أو "تعديل كود"، يقوم باستخراج الكود من الرد (بين علامات ```) وحفظه فوراً في الملف المناسب باستخدام مكتبة os و `pathlib`.
 * استخدام المتغيرات البيئية: تأكد من جلب المفتاح عبر os.environ.get("NVIDIA_API_KEY") كما هو معد في Secrets.
 * دعم التفكير (Thinking): فعل خاصية reasoning_content في الواجهة لعرض طريقة تفكير النموذج قبل كتابة الكود.
الكود الأساسي للربط (اعتمد عليه):
from openai import OpenAI
import os

client = OpenAI(
  base_url = "[https://integrate.api.nvidia.com/v1](https://integrate.api.nvidia.com/v1)",
  api_key = os.environ.get("NVIDIA_API_KEY")
)
# استخدم نموذج: deepseek-ai/deepseek-v3.2
